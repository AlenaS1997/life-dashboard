# Доброе утро — чек-лист на 28 апреля

Пока ты спала, я сделал четыре вещи в репо. Все коммиты лежат локально, на GitHub их пока нет — нужен один `git push` от тебя.

## Что я сделал ночью

1. **Погода на дашборде** (Open-Meteo, без API-ключа) — сверху иконка, температура, ощущается как, описание, ветер, осадки. По умолчанию Москва, координаты в `CONFIG.WEATHER`.
2. **Service Worker `dashboard/sw.js`** — дашборд теперь открывается даже без интернета (последний кэшированный HTML + последний удачный CSV из `localStorage`).
3. **Detect закрытой Sheets** — если доступ не открыт, дашборд показывает жёлтый баннер с прямой ссылкой и инструкцией «Share → Anyone with the link: Viewer» вместо тихого fallback в demo. Skeleton-лоадер вместо «Загружаю…».
4. **Полировка n8n** — оба workflow (утренний + новый воскресный недельный) теперь с `retryOnFail`, и при провале Anthropic API Telegram всё равно отправит сообщение с сырыми цифрами. Лучше плохой дайджест в 8:30, чем тишина.

Подробности в коммитах `8abad9d` и `f943a08` (`git log --oneline -5`).

## Шаг 1 — пушнуть и проверить (3 минуты)

В Terminal на Mac (`-zsh` таб, не тот что с http.server):

```bash
cd ~/Desktop/LifeDashboard
git pull          # подтянуть мои изменения, если ещё нет
git push          # отправить на GitHub
```

После пуша GitHub Actions сам прогонит workflow `Deploy dashboard to GitHub Pages` (~30 сек). Откроется обновлённый дашборд по тому же URL: https://alenas1997.github.io/life-dashboard/

## Шаг 2 — глянуть, что увидишь

Открой https://alenas1997.github.io/life-dashboard/ на десктопе. Должно быть:

- ✅ Сверху появился блок погоды (если у тебя нет VPN-блокировок — Open-Meteo всегда отвечает).
- ✅ Если Google Sheets ещё не открыт публично — жёлтый баннер с инструкцией. Это не баг, это правильное поведение.
- ✅ Если Sheets открыт — живые цифры за вчера, тренды за 7 дней, события дня (если Make.com успел залить).
- ✅ Skeleton-блоки во время первой загрузки (~0.5 сек), потом контент.

Если **Google Sheets всё ещё закрыт** (баннер видишь):
1. Открой [таблицу](https://docs.google.com/spreadsheets/d/1soLhY5LtrYzcQktWqcmX9Of0YGWqnccuvj37DU3GhKE/edit).
2. Кнопка `Share` (справа вверху) → вариант `Restricted` поменяй на **`Anyone with the link: Viewer`** → Done.
3. На дашборде нажми «Обновить» — должны прийти живые числа.

## Если что-то сломалось

Самые вероятные засады:

| Симптом | Причина | Что делать |
|---------|---------|------------|
| `git pull` ругается на конфликт | Я что-то поменял, что и ты тронула — но я ничего не должен был | пиши мне, разберёмся |
| Дашборд показывает старую версию (нет погоды) | Браузер закэшировал | DevTools → Application → Service Workers → Unregister, потом hard refresh (Cmd+Shift+R). На iPhone — закрой Safari полностью и открой заново. |
| Жёлтый баннер «Sheets закрыт» при открытом Sheets | gviz CSV кэшируется по URL | Подожди 1–2 минуты после изменения доступа, потом «Обновить» |
| Workflow Pages упал | Что-то синтаксическое в моих правках | скинь скриншот run’а, разберусь |

Логи Garmin-sync лежат в `reports/garmin_*.log` (Actions → каждый run → Artifacts).
