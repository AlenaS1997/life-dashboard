## Технические засады, которые пришлось разобрать

- **Garmin 429 на Mac IP** — ушло после второго retry; цикл логинов довёл до успеха. На GitHub Actions IP другой, проблем нет.
- **`'Garmin' object has no attribute 'garth'`** — версия `garminconnect` без аттрибута. Решено через прямой вызов модульного `garth.resume()` / `garth.save()` с тремя fallback-путями.
- **Дашборд показывал «Сон · 7 дней = 0.0 ч»** — Google Sheets ru_RU отдаёт CSV с запятой как десятичным разделителем; `Number("6,9") === NaN`. Решено `num()` хелпером, который чистит пробелы и меняет «,» на «.».
- **`.git/index.lock` permission denied** в песочнице — sandbox FUSE блокирует unlink. Решено через `mcp__cowork__allow_cowork_file_delete`.
- **`git push` из песочницы** — нет авторизации к GitHub. Push делает Алёна с Mac (keychain хранит PAT). Сама делает один `git push`, всё уезжает.
- **Pages 404 после первого workflow** — workflow прошёл раньше, чем Source был переключён на «GitHub Actions». Решено ручным «Run workflow» во второй раз.
- **GitHub Pages «Deploy from a branch» не умеет произвольную папку** — только `/` или `/docs`. Решено отдельным `.github/workflows/pages.yml` через `actions/upload-pages-artifact` с `path: ./dashboard`.
